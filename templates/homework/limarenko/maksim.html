Основные команды git

Git — это распределённая система контроля версий. Она позволяет разработчикам отслеживать изменения в исходном коде и сотрудничать с другими разработчиками над одним проектом.

Для начала работы с гит нам нужно его настроить. Для этого нам нужно всего 3 команды:

Настройка имени git config --global user.name "Ваше имя или логин"
Настройка email git config --global user.email "Ваш email"
Проверка конфигурации git config --list --show-origin

После чего в проекте делаем следующее:

инициализация проекта git init
добавление в индекс файлов git add
создать комит git commit -m "Первый коммит"
отправить проект на сервер git push
затянуть проект с сервера git pull
